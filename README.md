# Career Clarity

A 10-minute career discovery quiz that helps people figure out what they actually want to do next.

## Features

- **Comprehensive questionnaire** covering situation, skills, energy, work style, and deal-breakers
- **3 personalized career recommendations** based on your answers
- **Specific transition paths** from your current role
- **4-week action plans** with concrete tasks
- **Resource recommendations** (courses, books, communities)
- **Reality checks** — honest truth bombs about each career

## Careers Included

1. Product Manager
2. UX Researcher
3. Data Analyst
4. Startup Founder
5. Freelance Consultant
6. Health & Wellness Coach
7. Content Creator
8. Technical Writer

## Tech Stack

- React 18 (via CDN)
- Single HTML file (no build step required)
- Vanilla CSS

## Usage

Just open `index.html` in a browser. No server required.

## Development

This is a single-file React app. To make changes, edit `index.html` directly.
